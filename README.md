# Scroll-Driven Hero Section Animation

ASSINGNMENT

live link:https://capable-cassata-f223ba.netlify.app/

A smooth, performant scroll-driven hero section animation built with Next.js, Tailwind CSS, and GSAP.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)

## ✨ Features

- **Initial Load Animations**: Staggered letter-by-letter headline reveal with smooth fade-in
- **Scroll Parallax**: Image and text respond smoothly to scroll position with GSAP ScrollTrigger
- **Performance**: 60fps animations using GPU-accelerated transforms
- **Responsive**: Works seamlessly on all devices

## 🎨 Tech Stack

- Next.js 14 + React 18
- Tailwind CSS 3.4
- GSAP 3.12 with ScrollTrigger
- Performance-optimized with CSS transforms

## 📦 Deployment

```bash
npm run build && npm run export
```

Deploy the `out/` folder to any static hosting (GitHub Pages, Vercel, Netlify).
